Cara Mendapatkan UUID & Version

  1. Ekstrak atau buka file Addon (.mcpack / .zip) Anda.
  2. Cari file bernama manifest.json di folder utama, lalu buka dengan Text Editor.
  3. Cari bagian "header". (biasanya ada di bagian paling atas)
  4. Salin isi dari uuid dan version yang berada di dalam block header tersebut!
    Penting: Jika terdapat banyak UUID, abaikan UUID yang berada di dalam "modules" atau "dependencies". Server hanya mengenali UUID Header utama.
